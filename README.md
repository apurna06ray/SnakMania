🐍 SnakMania

SnakMania is a fun, interactive Snake game built with HTML, CSS, and JavaScript, featuring responsive design, sounds, start/restart buttons, and high score tracking.

📝 Project Structure
SnakMania/
├─ index.html       -> main HTML file
├─ styles.css       -> CSS styling & responsive design
├─ script.js        -> JavaScript game logic
├─ audio/           -> folder containing all audio files
├─ README.md        -> this file
🚀 Features

Classic Snake gameplay

Start & Restart buttons

Score & High Score tracking (localStorage)

Responsive design for mobile, laptop, and large screens

Background music & sound effects (move, eat, game over)

Increasing speed as you score more

🎮 Controls

Arrow Keys: Move the snake (Up, Down, Left, Right)

Start Game: Click "Start Game" button

Restart Game: Click "Restart Game" button

🛠 Built With:

HTML5

CSS3 (Flexbox + Grid + Media Queries)

JavaScript (DOM Manipulation + Event Listeners)
